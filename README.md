## 📦 What is Shasta?

Shasta plays .caar files. These files are archives that hold animations created for Apple’s Core Animation system. Normally, these are used on macOS computers, but Shasta lets you open and watch these animations with Windows.

This is helpful if you want to check animations without using a Mac, or if you want to show or archive Core Animation files. The app works with standard Core Animation bundles and offers smooth playback.

## 🛠️ Troubleshooting

If you have issues opening Shasta or playing animations, try these tips:

- Make sure your Windows is up to date.  
- Restart your computer and try running Shasta again.  
- Confirm you opened a valid `.caar` file made for Core Animation.  
- Check that your graphics driver is current.  
- Try running the app as administrator (right-click the shortcut, then “Run as administrator”).  
- If the animation lags, close other programs and try again.
